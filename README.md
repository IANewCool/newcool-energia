# NewCool Energia - Electricidad Chile

Portal educativo sobre energia electrica y tarifas en Chile.

## Demo

[https://energia.newcool.io](https://energia.newcool.io)

## Caracteristicas

- Buscador de distribuidoras electricas
- Comparador de tarifas
- Calculadora de consumo energetico
- Informacion de subsidios electricos
- Alertas de cortes de luz

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
